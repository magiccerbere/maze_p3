Welcome to "Freedom for Mac Gyver".

The goal of the game is to free Mac Gyver by finding the exit guarded by the watchman.
Three objects are necessary to put the watchman to sleep and get out.

The game runs on Python3.
Dependencies can be installed using the file requierements.txt :

pip install -r requirements.txt

The file to run is maze.py :

python3 maze.py

Have fun!
